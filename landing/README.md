# Clementime Landing Page

Professional marketing landing page for Clementime, built with React, Vite, and Tailwind CSS.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The landing page is served at [clementime.app](https://clementime.app).

### Building

```bash
npm run build
```

This creates an optimized production build in the `dist/` directory.

### Deployment Options

**GitHub Pages:**
- Build outputs to `dist/`
- Configure GitHub Pages to serve from this directory

**Cloudflare Pages / Vercel / Netlify:**
- Build command: `npm run build`
- Output directory: `dist`

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first styling
- **PostCSS** - CSS processing

## Features

- Dark Framer-style design
- Interactive platform comparison (macOS vs Web)
- Responsive layout
- SEO optimized with Open Graph and Twitter Card meta tags
- Smooth animations and transitions
- Glassmorphic UI elements
