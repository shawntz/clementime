# Clementime Landing Page

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The landing page is served at [clementime.app](https://clementime.app).

### Building

```bash
npm run build
```

This creates an optimized production build in the `dist/` directory.
